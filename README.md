# Web Animasi Interaktif - Animasi 1

Ini adalah salah satu proyek eksperimen animasi berbasis web yang saya buat. Proyek ini menampilkan implementasi animasi interaktif yang dirancang untuk memberikan pengalaman pengguna (*user experience*) yang menarik dan dinamis. Proyek ini nantinya akan diintegrasikan sebagai bagian dari galeri karya di website portofolio utama saya.

## 🚀 Live Demo
Kamu bisa melihat hasil instalasi dan tampilan langsung dari web animasi ini melalui tautan berikut:
👉 [Lihat Animasi di GitHub Pages](https://taufikprograming.github.io/Animasi-1/)

## 🛠️ Teknologi yang Digunakan
* **HTML5:** Untuk menstrukturkan elemen-elemen halaman web.
* **CSS3 / JavaScript:** Digunakan untuk membangun logika, transisi, dan pergerakan animasi agar terlihat halus dan interaktif.

## 📂 Struktur Proyek
* `index.html` - Halaman utama penampung animasi.
* `Asset/` - Folder penyimpanan aset pendukung proyek.
* `.vscode/` - Konfigurasi lingkungan teks editor.
